# Resto Control v2 — запись 012

## Главная: шапка, оплата и вкладки ресторанов

### Исправление
Верх главной страницы был разрозненным: вкладки ресторанов, кнопка добавления ресторана, поиск и бейдж оплаты визуально распадались.

### Новая структура
Верхняя строка:
- слева приветствие владельца;
- по центру бейдж оплаты;
- справа поиск, уведомления, профиль и выход.

Ниже отдельная рабочая строка ресторанов:
- вкладки ресторанов идут в один ряд;
- активный ресторан выделен тёмной вкладкой;
- кнопка «+ Добавить ресторан» стоит в этой же строке после разделителя;
- строка расположена над основными блоками главной страницы.

### Правило
Рестораны владельца не должны быть разложены под заголовком отдельными случайными кнопками. Это единый переключатель контекста ресторана.
