# Resto Control v2 — запись 018

## Мобильная версия — первая страница сотрудника

### Экран
- Первый экран мобильной версии называется **«Обзор»**.
- Это главная рабочая страница сотрудника после входа.

### Что убрано
- отдельный блок фотоотчётов;
- блок быстрых действий;
- дублирующие кнопки;
- кнопка «Домой» в нижнем меню.

### Верх экрана
- приветствие;
- должность и ресторан;
- кнопка уведомлений;
- кнопка меню с тремя точками.

### Основной блок
- статус смены;
- открыта / не открыта;
- время смены;
- кнопка открыть / закрыть смену.

### Блок «Что нужно сделать»
Карточки могут включать:
- чек-лист открытия;
- задачи;
- инвентаризацию — **только если она назначена**;
- стоп-лист — только для старших;
- **план зала** — вместо блока «План зала».

### Срочные уведомления
- отдельный блок важных уведомлений;
- детальный список открывается по кнопке уведомлений или по кнопке «Все».

### Нижнее меню
- Обзор;
- Задачи;
- центральный **+**;
- Чек-листы;
- Ещё.

### Центральная кнопка +
Центральная кнопка создаёт **техническую заявку**.

Примеры заявок:
- не работает лампа;
- проблема с оборудованием;
- нехватка расходников;
- клининг;
- поломка мебели.

### Меню «Ещё» / три точки
Включает:
- Знакомство с компанией;
- Корпоративная жизнь;
- Постоянные гости;
- База знаний;
- Поддержка;
- Выход.

### Роли
- должность «Клининг» не используется;
- вместо неё используется **Клининг**.
