# Live-журнал Resto Control v2

## Запись 006 — Исправление ширины таблицы сотрудников

Решение: таблица сотрудников не должна быть зажата карточной обёрткой или постоянной правой колонкой.

Фиксация:
- таблица сотрудников занимает всю доступную ширину центральной области;
- фильтры и таблица не заключены в одну большую карточку-обёртку;
- форма добавления сотрудника не забирает ширину у таблицы, а переносится ниже рабочей таблицы;
- кнопка действия в строке не должна переноситься по буквам;
- интерфейс обязан нормально читаться при масштабе браузера 100%.
