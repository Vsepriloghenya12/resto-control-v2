# Live-журнал Resto Control v2

## Запись 005 — Масштаб и плотность интерфейса

Фиксируем правило: интерфейс должен нормально выглядеть при стандартном масштабе браузера 100%, без необходимости уменьшать страницу до 80%.

Что исправлено для страницы «Сотрудники»:

- общий layout стал компактнее;
- sidebar уменьшен по ширине;
- main padding уменьшен;
- верхняя шапка остаётся одной собранной строкой;
- карточки показателей стали компактнее;
- таблица сотрудников стала плотнее;
- правая панель добавления сотрудника стала компактнее;
- при недостаточной ширине правая панель переносится ниже таблицы, чтобы данные не обрезались.

Правило на будущее: если экран требует уменьшения масштаба браузера, значит дизайн сделан неправильно. Нужно адаптировать сетку, размеры, переносы и плотность данных под обычный 100% масштаб.
