# Live-журнал Resto Control v2

## Запись 007 — Страница «Задачи»

Страница «Задачи» предназначена для создания, редактирования и назначения рабочих задач. Она не дублирует главную страницу: контроль просрочек, проблем и сегодняшнего выполнения остаётся на главной панели владельца.

### Зафиксированная логика

- задача может назначаться конкретному сотруднику или должности;
- у задачи есть название, описание, срок выполнения, статус, активность и признак обязательного фотоотчёта;
- если фото выполнения обязательно, сотрудник не сможет закрыть задачу без фото;
- страница показывает список задач слева и редактор выбранной задачи справа;
- оперативная инфографика на этой странице не нужна.

### Статусы

- Не начата;
- В работе;
- Выполнена;
- Просрочена.

### Поля задачи

- название задачи;
- описание задачи;
- кому назначить;
- тип назначения: должность или конкретный сотрудник;
- дата выполнения;
- время выполнения;
- нужно фото выполнения;
- активна;
- дополнительная информация.

### Визуал

Страница сделана в том же рабочем стиле, что «Чек-листы»: левый список и правый редактор, без лишней инфографики, без demo-подписей, без декоративных блоков и без обёрток, которые режут полезную ширину.
