# Live-журнал Resto Control v2

## Запись 008 — План зала / Брони

Страница нужна владельцу и управляющему для настройки залов, столов, посадочных мест и работы с бронями.

## Зафиксированная логика

- Вкладка называется «План зала / Брони».
- Без картинки помещения, drag-and-drop и визуальной схемы.
- План зала — это структура: зал → столы → посадочные места → брони.
- Владелец задаёт залы, количество столов и посадочные места.
- Администратор видит, где что забронировано, кто пришёл и где гости уже сели.
- «Пришли по брони» и «Гости сели» — разные этапы.

## Что есть на странице

- Переключатель «План зала / Брони».
- Фильтры: дата, поиск гостя/телефона, статус.
- Список залов с количеством столов, посадочных мест и статусами.
- Таблица столов выбранного зала.
- Правая панель выбранного стола/брони.
- Правая панель «Настройки зала».

## Настройки зала

Кнопка «Настройки зала» открывает правую рабочую панель, а не отдельную страницу.

В настройках:

- название зала;
- количество столов считается автоматически;
- количество посадочных мест считается автоматически;
- активен/выключен;
- список столов выбранного зала;
- название стола;
- количество мест;
- статус стола;
- добавить стол;
- сохранить настройки;
- удалить зал.

## Статусы столов

- Свободен
- Забронирован
- Пришли по брони
- Гости сели
- Недоступен

## Что не добавляем

- картинку помещения;
- перетаскивание столов;
- координаты столов;
- рисование зон;
- финансовые показатели;
- лишнюю инфографику.

## Статус

Страница встроена в проект. Пункт меню «План зала / Брони» открывает новый экран.
