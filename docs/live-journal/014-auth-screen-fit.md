# Resto Control v2 — запись 014

## Правка первой страницы входа

### Проблема
Первая страница входа была выше экрана браузера и появлялся вертикальный скролл. Левая брендовая часть обрезалась снизу.

### Решение
- Экран авторизации на desktop теперь занимает высоту viewport.
- Внешний скролл у первой страницы убран.
- Высота основного auth-блока считается от высоты экрана.
- Левая брендовая колонка стала компактнее: уменьшены вертикальные отступы, расстояния между блоками и форма.
- Форма входа и регистрации ужаты по высоте без изменения логики.
- На мобильном варианте скролл сохранён, чтобы форма корректно помещалась на маленьких экранах.

### Правило
Первая страница входа должна помещаться в один экран desktop-браузера без вертикального скролла и без обрезания нижних элементов.
