# Запись 005 — Правило верхней шапки страниц

Верхняя шапка страницы должна быть единым собранным рядом.

Правило:
- слева находится заголовок текущего раздела и короткое описание;
- справа находятся поиск, уведомления, профиль и выход;
- левый блок и правый блок выровнены по одному визуальному центру;
- нельзя делать отдельный верхний блок действий, а заголовок опускать ниже;
- на каждой странице заголовок должен быть один раз, без дублей.

Исправлено:
- страница «Сотрудники» больше не рисует отдельный заголовок ниже верхней панели;
- заголовок «Сотрудники» встроен в общий topbar;
- поиск, уведомления, профиль и выход выровнены в одном ряду с заголовком;
- кнопка добавления сотрудника не возвращалась в верхнюю шапку.
