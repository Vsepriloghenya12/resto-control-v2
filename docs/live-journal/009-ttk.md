# Resto Control v2 — запись 009

## Страница «ТТК»

### Название вкладки
- Вкладка называется **«ТТК»**.
- Название **«ТТК и рецептуры»** не используется.

### Назначение страницы
Страница нужна владельцу/управляющему для ведения полной карточки блюда или товарной позиции.

### Структура страницы
- слева — группы / категории;
- по центру — список позиций;
- справа — карточка выбранной позиции.

### Что есть в карточке позиции
#### Основное
- фото;
- наименование;
- единица измерения;
- цена;
- группа;
- тэг;
- описание.

#### Технология
- раскладка;
- время приготовления;
- выход готового блюда.

#### Добавки и пары
- добавки;
- гастрономические пары.

#### КБЖУ
- ккал;
- белки;
- жиры;
- углеводы.

#### Продажа
- доступно на вынос;
- доступно для онлайн-заказа;
- скидки применяются.

#### Ограничения и контроль
- маркировочный товар;
- нельзя взять без пика / сканирования;
- подакцизный товар.

### Что убрано из ТТК
- статус позиции;
- признак «в стоп-листе»;
- любые блоки, связанные с временным стоп-листом.

### Групповые операции
Доступны операции по всей группе:
- включить / выключить скидки;
- включить / выключить онлайн-заказ;
- включить / выключить на вынос;
- назначить / снять маркировку;
- включить / снять признак подакцизности;
- запретить / разрешить выдачу без пика / сканирования.

### Важное правило
ТТК — это постоянная карточка блюда/позиции. Стоп-лист не показывается на этой странице и не управляется отсюда.
